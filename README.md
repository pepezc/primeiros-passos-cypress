# My Cypress Automation

This project utilizes Cypress to automate the testing of a login feature and a user update feature.
Cypress is a JavaScript-base end-to-end testing framework that allows developers to write tests and
execute them in a real browser.

## Installation
```bash
npm install
```
>**NOTE**
>
>It's necessary to have **node** installed

## Running
```bash
# You open Cypress UI
npm cypress open

# Run the automation by command line
npm cypress run
```
